# Visuelle-Flugobjekt-Odometrie
im Ordner data sind die Projektdateien für Google Earth Studio für TestSeup1 und TestSetup2 zu finden.
Eine Datei gt.txt ist ebenfalls jeweils enthalten.
Diese Datei kann auch mit utils/ges_dl_plot.py erzeugt werden.
Hierzu ist der Pfad in der Datei entsprechend anzupassen.
Im Ordner utils sind test-Skripte zur Auswertung.
Die Datei vo.py enthält die Implementierung von Algorithmus 1 und vo2.py die Implementierung von Algorithmus 2.
